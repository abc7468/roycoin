## Folders
- explorer
  - templates
    - pages : 라우팅 될 **페이지**
    - partials : 페이지에 로드될 **블락** 

- blockchain
  - block : block의 Struct와 Method
  - chain : blockchain의 struct와 Method

- rest : restAPI

- cli : terminal에서 flag설정을 해주는 cli 작성

- utils : 구현에 도움을 주는 helper 코드

- db : persistance를 위한 함수(Bolt DB)
